# 🤱 Калькулятор декретних для ФОП — 2026

Простий онлайн-калькулятор для розрахунку допомоги по вагітності та пологах (декретних) для фізичних осіб-підприємців в Україні.

👉 **[Відкрити калькулятор](https://podstava.github.io/dekretny-fop/)**

---

## Що рахує

- Середньоденну допомогу на основі бази ЄСВ за 12 місяців
- Загальну суму за 126 / 140 / 180 днів відпустки
- Автоматично застосовує мінімальний та максимальний ліміт (Закон №1105)
- Попереджає про обмеження при страховому стажі менше 6 місяців

## Ключові цифри 2026

| Параметр | Значення |
|---|---|
| Мінімальна зарплата | 8 647 грн |
| Мінімальна база ЄСВ | 8 647 грн |
| Максимальна база ЄСВ | 172 940 грн (×20) |
| Мінімальні декретні (126 днів) | ~35 792 грн |
| Максимальні декретні (126 днів) | ~715 848 грн |
| Розрахунковий період | 365 днів |

## Як користуватись

1. Обери тривалість відпустки (126 / 140 / 180 днів)
2. Вкажи кількість місяців зі сплаченим ЄСВ за останній рік
3. Введи місячну базу ЄСВ (натисни ℹ️ якщо не знаєш що це)
4. Натисни «Розрахувати суму»

## Важливо

> Калькулятор є довідковим інструментом. Для подачі офіційної заяви-розрахунку до ПФУ рекомендується звернутись до бухгалтера або на портал [portal.pfu.gov.ua](https://portal.pfu.gov.ua).

## Нормативна база

- Закон України «Про загальнообов'язкове державне соціальне страхування» №1105
- Порядок призначення допомоги по вагітності та пологах №1266
- Закон України «Про збір та облік єдиного внеску» №2464
